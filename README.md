# 数学计算秘籍本 📒

给三年级孩子整理的数学「计算秘籍」讲义,一个秘籍一页,A4 打印友好。讲义根据真题(原始题集 / 错题集)整理而来。

🔗 **在线浏览**(GitHub Pages):https://bluebu.github.io/math/

## 目录结构

| 目录 | 说明 | 是否发布到 Pages |
| --- | --- | --- |
| `docs/` | 发布内容:`index.html`(秘籍本首页)+ 各秘籍讲义 | ✅ 发布 |
| `原始题集/` | 原始资料:试卷 PDF(5·3 全优卷 三年级下册 RJ) | ❌ 不发布 |
| `错题集/` | 原始资料:错题照片 | ❌ 不发布 |

## 启用 GitHub Pages

仓库 **Settings → Pages → Build and deployment**,Source 选 **Deploy from a branch**,Branch 选 `main`、文件夹选 **`/docs`**,保存。稍等片刻即可通过上面的链接访问。
