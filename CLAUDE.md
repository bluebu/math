# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 这是什么

这不是代码仓库，而是一份**小学三年级数学（人教版 RJ，下册）学习资料集**。内容面向一名三年级学生，用于做题、讲解和整理错题。没有构建 / 测试 / lint 流程，不要凭空创造这类命令。

所有沟通和讲解使用中文（见全局配置）。讲解对象是三年级小学生，方法必须符合该学段认知：用算术方法而非代数方程，避免超纲概念，步骤清晰、配合生活化的语言。

## 目录结构与命名约定

- `原始题集/` —— 题目来源（试卷 PDF）。
  - `53/` 指「5·3 全优卷 小学数学 三年级下册 RJ」这一套教辅。
  - 文件如 `期末模拟（一）.pdf` … `期末模拟（五）.pdf`，是整套期末学业质量评价模拟卷。每份卷子约 70 分钟、满分 100 分。
- `错题集/` —— 做错的题目，以照片（JPG）形式收集。
  - 文件名为数字编号（如 `317.JPG`），`-2` 后缀（如 `317-2.JPG`）表示同一题的另一张照片/版本。

读取资料：PDF 用 Read 工具的 `pages` 参数按页读取；错题照片直接用 Read 读取图片识别题目。

## 三年级下册（人教版）涉及的主要题型

从现有材料看，覆盖：轴对称图形、小数的初步认识（如 0.4 的含义）、除法竖式（含商中间/末尾有 0、有余数）、周长与面积的比较和计算、长方形与正方形、年月日与日期推算（如三伏天天数）、周期与归一问题。讲解时按这些主题选用对应的标准解法。

## 典型工作流

- **讲题**：先读懂题目（必要时读取对应 PDF 页或错题图片），给出三年级能理解的解题思路和最终答案，关键步骤说明「为什么」。
- **整理错题**：从 `原始题集` 的 PDF 中定位题目，或根据 `错题集` 的照片，分析错因、给出正确解法。新增错题照片沿用「数字编号 + 可选 `-N` 后缀」的命名。
