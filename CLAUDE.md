# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## 这是什么

这不是普通代码仓库，而是一份**小学三年级数学（人教版 RJ，下册）学习资料 + 讲义站点**。核心产物是一套给三年级孩子的「计算秘籍」讲义（HTML，A4 打印友好），由原始真题整理而来，并通过 GitHub Pages 发布。没有构建 / 测试 / lint 流程，不要凭空创造这类命令。

所有沟通和讲解使用中文（见全局配置）。讲解对象是三年级小学生，方法须符合该学段认知：用算术方法而非代数方程，避免超纲，步骤清晰、配合生活化语言。教学风格：把题型提炼成一句好记的「计算秘籍」（如 求经过天数 ＝ 最后一天 − 第一天 + 1），再用秘籍解具体题、点出易错点。

## 目录结构与发布模型

仓库 `git@github.com:bluebu/math.git`，站点 https://math.hi-ruby.com/ （自定义域名，由 `docs/CNAME` 记录；DNS 用 CNAME 指向 `bluebu.github.io`）。GitHub Pages 配置为「`main` 分支 / `docs` 文件夹」，因此**只有 `docs/` 会被发布**。

- `docs/` —— ✅ **发布内容**。
  - `index.html` —— 秘籍本首页（秘籍列表，点击进入各讲义；含脚本按记录日期自动排序、统计数量）。
  - 各秘籍讲义，如 `求经过天数-计算秘籍.html`；每页顶部有「‹ 返回秘籍本」链接（`back screen-only`，打印时隐藏）。
  - `.nojekyll` —— 关闭 Jekyll，按原样发布。
- `原始题集/` —— ❌ **原始资料，不发布**。试卷 PDF；`53/` 是「5·3 全优卷 小学数学 三年级下册 RJ」，`期末模拟（一）…（五）.pdf` 为 5 份期末模拟卷（各 70 分钟、满分 100）。
- `错题集/` —— ❌ **原始资料，不发布**。做错题目的照片（JPG），文件名为数字编号（如 `317.JPG`）。

读取资料：PDF 用 Read 的 `pages` 参数按页读取；错题照片直接 Read 识别题目。

## 新建一个秘籍讲义

1. 从 `原始题集` / `错题集` 的真题里确定题型与对应「秘籍」。
2. 在 `docs/` 下新建讲义 HTML（A4 打印友好：`@page size:A4`、`print-color-adjust:exact`、系统中文字体；结构：秘籍公式框 + 生活化「为什么」 + 易错点 + 真题例题 + 小工具 + 练一练带答案；顶部加「‹ 返回秘籍本」`back screen-only` 链接）。
3. 在 `docs/index.html` 复制卡片模板新增一行（href、emoji、名称、公式、单元标签、记录日期 `data-date`）。
4. 用 `open <文件>` 在浏览器预览。

## 三年级下册（人教版）涉及的主要题型

轴对称图形、小数初步认识（如 0.4 的含义）、除法竖式（商中间/末尾有 0、有余数）、周长与面积、长方形与正方形、年月日与日期推算（如三伏天天数）、周期与归一问题。讲解时按这些主题选用对应的标准解法。
