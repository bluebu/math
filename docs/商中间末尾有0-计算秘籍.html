商中间、末尾有 0 · 计算秘籍

三年级下册 · 除法竖式
计 算 秘 籍
哪一位「不够除」，商就在那位写 0
把这一位的数和除数比一比——比除数 小 就是「不够除」，商写 0 占位，再接着往下除。

为什么不够除要写 0 ？（分糖故事）

🍬 把 812 颗糖，平均分给 4 个小朋友
糖分成三种：100 颗一堆、10 颗一捆、1 颗散的，从大往小分。
• 百位：8 堆（800 颗）分给 4 人，每人 2 堆，正好分完 → 商百位 2。
• 十位：只有 1 捆，4 个人，1 捆不够每人分 1 捆（不够除） → 每人 0 捆 → 商十位写 0！把这 1 捆拆成 10 颗。
• 个位：原来 2 颗 ＋ 拆开的 10 颗 ＝ 12 颗，分给 4 人，每人 3 颗 → 商个位 3。
每人分到 2 堆 0 捆 3 颗，也就是 812 ÷ 4 ＝ 203。中间的 0 表示「十位上每人分到 0 捆」，必须写，不能跳过！

看懂两种「0」长在哪里

812 ÷ 4 ＝ 203
商中间有 0：是 中间那一位（十位）不够除
842 ÷ 4 ＝ 210 …… 余 2
商末尾有 0：是 最后一位（个位）不够除

真题①：8□2÷4，商中间有 0 且没有余数

  • 百位：8 ÷ 4 ＝ 2，正好分完没剩。
  • 要商中间（十位）是 0：就要十位的 □「不够除 4」 → □ 比 4 小 → □ 可以是 0、1、2、3。
  • 再加「没有余数」：把这几个一个个试除，挑能除尽的 → 812÷4＝203 ✓、832÷4＝208 ✓。（802、822 都余 2，划掉）
  • 所以 □ 填 1 或 3。

真题②：8□2÷4，商末尾有 0

  • 要商末尾（个位）是 0：就要个位「不够除 4」。
  • 个位本来是 2（比 4 小），但前提是 十位要先除尽、别留余数下来——不然余数带到个位，个位就够除了。
  • 所以十位 □÷4 要正好除尽 → □ 是 4 的倍数 → □ ＝ 0、4、8。
  • 验证：802÷4＝200…2、842÷4＝210…2、882÷4＝220…2，商末尾都是 0。所以 □ 填 0、4、8。
💡 这一问只要求「商末尾有 0」，没要求没有余数，所以三个答案虽然都余 2，也都算对。

易错点（最容易丢分的地方）

⚠️ 不够除时，0 千万不能漏！
• 不够除时 0 要写在商里占位，绝不能跳过 —— 漏了 203 就会写成 23，整道全错。
• 分清两种：「商中间有 0」看 中间那位够不够除；「商末尾有 0」看 个位够不够除。
• 看清题目要不要「没有余数」：要，就得在候选里挑能除尽的；不要，有余数也可以。
✅ 检验秘籍：商 × 除数 ＋ 余数 ＝ 被除数。如 203×4＝812 ✓；210×4＋2＝842 ✓。

练一练（先想：让哪一位「不够除」）

  1. 计算 6□4÷6，要使商中间有 0，□里可以填 。
  2. 计算 9□5÷9，要使商末尾有 0，□里可以填 。
  3. 计算 8□0÷4，要使商中间有 0 且没有余数，□里可以填 。
  4. 808÷4 ＝ ；780÷6 ＝ 。（算一算，看看 0 长在哪）
参考答案：
① 6÷6＝1 正好；要十位商 0 就让 □ 比 6 小 → □ ＝ 0、1、2、3、4、5。（如 604÷6＝100…4，商中间是 0）
② 要个位商 0，就让十位除尽 → □ 是 9 的倍数 → □ ＝ 0、9。（905÷9＝100…5、995÷9＝110…5）
③ 十位 □ 比 4 小才会商中间 0，再挑能除尽的 → □ ＝ 0、2。（800÷4＝200、820÷4＝205）
④ 808÷4＝202（商中间有 0）；780÷6＝130（商末尾有 0）。
